Configurar SSO OAUTH
 

Administración

  • Debes hacer clic para acceder a los ajustes de la plataforma.
Parámetros Generales

Aquí encontraremos todas las configuraciones que nos entrega la plataforma.

Debes ingresar la palabra Omniauth para encontrar la configuración.

  • Desplegamos la lista y seleccionamos General.

Luego de activa la opción, daremos clic en el botón guardar.

Administración

  • Debes hacer clic para acceder a los ajustes de la plataforma.

Debes ingresar la Integraciones para encontrar la configuración correcta.

  • Seleccionamos la configuración Integraciones
  • Aquí encontraremos diferentes integraciones, pero en este caso debemos seleccionar Single Sign On
Habilitar link SSO Habilita un link directo al acceso corporativo
Nombre Link SSO Nombre que tendrá el link directo.

Tenemos 3 formar de realizar la autenticación con Oauth. Debes seleccionar una forma de autenticarte

Google OAuth

Se ingresan el o los dominios a autenticar en la configuración bajo Autenticación de Google.

Office365 OAuth

Se ingresan el o los dominios a autenticar en la configuración bajo Autenticación de Office365.

Estrategia OAuth personalizada Nombre de la estrategia personalizada
Dominios a autenticar Dominios de correo electrónico que se utilicen en este método de autenticación, si es más de uno, separar con punto y coma (;)
client_id Identificador de BUK según el proveedor de autenticación. Solicitar a administrador.
client_secret Cliente secreto que se utilizará para autenticar la aplicación de buk en el servidor de OAuth.
authorize_url Dirección URL donde se realizarán las solicitudes de autenticación.
token_url Dirección URL donde se solicitarán los token para las sesiones de autenticación.
userinfo_endpoint Dirección URL donde se obtiene la información del usuario logeado.
tenant_id (Opcional) Código de grupos de usuarios que se pueden usar para autentificar.
logout_url Dirección URL donde se redirigirá al usuario para cerrar su sesión tanto en BUK como en la plataforma personalizada.
Enviar Email completo a Login Envío de email con el dominio a la pantalla del login del proveedor de atenticación

Luego de configurar cada uno de los campos solicitados, daremos clic en el botón guardar.
