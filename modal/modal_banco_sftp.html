Administración

  • Debes hacer clic para acceder a los ajustes de la plataforma.

Debes ingresar la palabra Integraciones para encontrar la configuración correcta y seleccionamos Integraciones

Envío de archivos de Bancos

  • Aquí seleccionamos esta opción para comenzar con la configuración

Seleccionar Empresa

  • La configuración debes realizarla por empresa, en este caso solo figura una empresa.
Bacup URL Debes especificar si tu servidor es SFTP o FTP y agregar tu url compuesta por el tipo de servidor, usuario, dirección ip o dns y por último el puerto de conexión, ejemplo: sftp://usuario@127.0.0.1:22
Backup Password Aquí debes ingresar la contraseña de tu servidor
Backup Path Esta ruta hace referencia en donde sera almacenado el archivo en tu servidor Ej ('/etc/tmp/')
Encriptación Activa Si necesita que tu archivo viaje encriptado, debes activar esta casilla
Llave Pública Las llaves SSH o publicas se usan para autenticación en servidores remotos. Si utilizas SSH, no es necesaria la contraseña
Recipiente de la llave pública (opcional) Es un correo de respaldo o identificador del dueño de la Llave publica

Luego de configurar cada uno de los campos solicitados, daremos clic en el botón guardar.

Damos clic en el logo de BUK.

Seleccionar Mes

  • Seleccionamos el mes correspondiente al archivo de banco que queremos enviar al servidor

Seleccionamos los tres puntos para abrir las opciones

Enviar banco

  • Una vez que hagamos clic comenzara el proceso, solo debemos esperar.
  • Aquí podemos ver en la parte superior derecha como va el estado del envió
  • Ya podemos ver en la parte superior derecha el mensaje de éxito!
