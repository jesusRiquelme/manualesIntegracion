Creación de token y conexión
 

Administración

  • Debes hacer clic para acceder a los ajustes de la plataforma.

Debes ingresar la palabra Accesos API para encontrar la configuración correcta.

Accesos API

Haz clic aquí para ingresar a la creación del token

Clic aquí para que comencemos a crear el token.

  • En este punto, es momento de revisar cada uno de los permisos que requiera el endpoint que desees utilizar.

Luego de configurar los permisos para los endpoint que utilizaremos, daremos clic en el botón guardar.

Damos clic en el icono del ojo para que el token sea desbloqueado y poder copiarlo.

Copiamos el texto y lo llevaremos a la pagina de apidocs. Ha esta pagina llegamos ingresando la palabra apidocs en la url del cliente ejemplo: demo.buk.cl/apidocs.

Debemos hacer clic en Autorizar para que ingresemos nuestro token copiado anteriormente.

  • Debemos ver que el país este correcto

Pegamos nuestro token creado.

Hacemos clic en Authorize

Volvemos hacer clic en autorizar. Esto es para poder validar que figuremos logeados en apidocs.

  • Podemos ver que el bóton ya dice logout (cerrar sesión) y que en el campo value figura nuestro token.

Si aún te encuentras con dudas, puedes escribirnos en nuestro canal de slack haciendo clic en el icono o también puedes revisar nuestra documentación en el logo de google drive
